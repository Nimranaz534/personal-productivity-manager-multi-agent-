# Personal Productivity Manager (Multi-Agent)

An AI-powered productivity manager built using n8n multi-agent framework.

## Features
- Task Agent — manages and saves tasks automatically
- Note Agent — captures and stores notes
- Reminder Agent — sets and tracks reminders
- Groq LLM brain for each agent with conversation memory
- Triggered via chat message

## Tools Used
- n8n (workflow automation)
- Groq (LLM)
- Google Sheets (data storage)
